# monkmail

A lightweight Node.js library for Telegram notifications, SMTP emails, and GitHub webhook handling.

## Packages

- **[packages/app](./packages/app)** - Core monkmail library (npm package)
- **[packages/web](./packages/web)** - Documentation website

## Quick Install

```bash
npm install monkmail
```

See [packages/app/README.md](./packages/app/README.md) for full documentation.
